### 步骤：
1. **在 Web 上**（`http://localhost:3000`）：
   - 登录
   - 添加活动到购物车
   - 进行预订

2. **在 Mobile 上**（Expo Go）：
   - 用同一个账号登录
   - 应该能看到相同的数据
   - 预订历史应该同步

3. **数据同步测试**：
   - Web 和 Mobile 使用同一个 Supabase 后端
   - 数据会实时同步

## ⚠️ 如果看到错误

### 常见问题：

#### 1. 白屏或空白页面
- 检查 PowerShell 窗口是否有错误信息
- 在 Expo Go 中点击 "Reload"

#### 2. 网络错误
- 确保手机和电脑在同一 WiFi
- 或使用 `--tunnel` 模式（已使用）

#### 3. 无法加载数据
- 检查 `.env` 文件中的 Supabase 配置
- 确保 Supabase 服务正常运行
